After finished steps before 6th:

1.
Start ZK ensemble
$ cd ~/apache-zookeeper-3.8.5-bin/$(hostname)
$ ~/apache-zookeeper-3.8.5-bin/bin/zkServer.sh start zoo-base.cfg

close ZK ensemble
$ cd ~/apache-zookeeper-3.8.5-bin/$(hostname)
$ ~/apache-zookeeper-3.8.5-bin/bin/zkServer.sh stop zoo-base.cfg

0. compile all .java files:
~/DS/A3$ javac -cp ~/apache-zookeeper-3.8.5-bin/lib/*:. $(find zk/zk -name "*.java")

1. ssh to each server, startup ZooKeeper ensemble
cd ~/apache-zookeeper-3.8.5-bin/$(hostname)
~/apache-zookeeper-3.8.5-bin/bin/zkServer.sh start zoo-base.cfg

3. To connect to the ZooKeeper ensemble using client command line interface CLI
 back to main server
$ cd ~
$ ~/apache-zookeeper-3.8.5-bin/bin/zkCli.sh -server tr-open14.cs.mcgill.ca:21804, tr-open-16.cs.mcgill.ca:21804, tr-open17.cs.mcgill.ca:21804

4. create tasks, workers node under /dist04
After start the ZooKeeper, in the command line, input:
create /dist04 ""
create /dist04/tasks ""
create /dist04/workers ""

5. set ZOOBINDIR
export ZOOBINDIR=[home_dir]/apache-zookeeper-3.8.5-bin/bin

6. run client
$ chmod +x runclnt.sh
$ ./runclnt.sh 500000

7. list your JAVA processes
$ ps -u $USER | grep java
